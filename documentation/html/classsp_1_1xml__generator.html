Detailed Description

generates the XML file

The generator receives the syntax tree of a policy and produces a XML output of this policy. It requests the corresponding sp::base_policy of the driver. For the code generation the TinyXML-2 library is used. At first a XML document is created. The generator creates XML nodes which are attached to this document. At last the XML document will be written to the stream and returned.
