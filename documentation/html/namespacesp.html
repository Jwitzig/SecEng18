◆ log_level

enum sp::log_level

Available log levels.

Enumerator
FATAL 

Errors that should never occure. this indicates an unexpected behaviour of the compiler.

ERROR 

Errors in filenames, syntactic and semantic errors in the policy files, violations of the type system.

WARNING 

Warning messages for inconsistent values in the policies.

INFO 

Basic messages of the compiler, such as the current state of the compiling process.

DEBUG 

Detailed debugging messages of the execution steps of the compiler.
