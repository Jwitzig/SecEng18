Detailed Description

Prepares the base policy for C++ output generation.

This class processes the base policy. It uses the parsed data and transforms it into suitable containers. It creates:
1.) Conditions/Primitives definitions and their function body
2.) The type definitions for all policy components (e.g. map with name x maps element y on element z)
