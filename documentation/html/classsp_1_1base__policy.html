Detailed Description

contains a base policy. A sp::base_policy represents a base policy. It is a supporting structure for the sp::worker to collect and use information to check the semantic of the specification. Furthermore, the generator uses it in order to differentiate between different component types. It contains all important parts which make up a base policy.
