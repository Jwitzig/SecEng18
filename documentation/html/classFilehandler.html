Detailed Description

Generates the C++ output. Uses data provided by cpp_generator and cpp_base_generator.

This class takes all the Information from the generator class.
It creates the .cpp and .hpp files.
Includes and and other important constructs are also added to the C++ files.

Following files will be created (Header/Source):

  1. Policy Layer hpp/cpp (Authorization sheme)
  2. Processing Layer hpp/cpp (conditions and primitives)
  3. Data Layer hpp (State Space and Extension Vector)
  4. Components Definition hpp
